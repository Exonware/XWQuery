# xwui — API Reference

**Last Updated:** 09-Feb-2026

API reference for xwui (output of GUIDE_15_API). See [REF_22_PROJECT.md](REF_22_PROJECT.md) and [REF_13_ARCH.md](REF_13_ARCH.md).

## Scope

- **Public surface:** TypeScript/JS component library; Custom Elements and optional framework wrappers (Vue, Svelte, Angular, Stencil, Stimulus).
- **Entry points:** Component modules under `src/` (styles, tokens, utils); build output (Vite library mode); examples in `examples/`.
- **Firebase frontend parity:** UI components that integrate with xwauth, xwstorage/xwchat, and xwapi/xwai-server are described in [REF_22_PROJECT.md — Firebase Frontend Parity](REF_22_PROJECT.md#firebase-frontend-parity). API patterns (auth flows, data bindings, API calls) follow those integrations.
