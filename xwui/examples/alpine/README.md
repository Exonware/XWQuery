# XWUI + Alpine.js

XWUIButton (Custom Element) with **Alpine.js** (CDN). Minimal setup.

## Run

Serve this folder (e.g. `npx serve -l 3001`), then open the URL.

```bash
cd examples/alpine
npx serve -l 3001
```

Assets are copied from `../common/`; see [../README.md](../README.md) to refresh from common.
