# XWUI + Fresh

Static demo for [Fresh](https://fresh.deno.dev/) (Deno). In a Fresh app, serve XWUI assets from static and use `<xwui-button>` in islands or HTML.

**Run:** From this folder: `npx serve -l 3022` (or `deno task start` in a full Fresh project). Open http://localhost:3022.

**Setup:** Run `node examples/sync-common.js` from repo root.
