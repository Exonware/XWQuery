# XWUI + Qwik

XWUIButton (Custom Element) in a **Qwik**-style app (Vite). Uses plain Vite (no qwikVite) so the build produces a runnable dist with index.html; XWUIButton works the same.

## Setup

```bash
cd examples/qwik
npm install
```

Copy common assets into `public/`: `xwui-button.mjs` and `xwui/` from `examples/common/`, or run `node examples/sync-common.js` from repo root.

## Run

```bash
npm run dev
```

Or build and preview: `npm run build` then `npm run preview`.
