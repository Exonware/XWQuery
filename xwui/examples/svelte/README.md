# XWUI + Svelte

XWUIButton (Custom Element) in a **Svelte 5** app (Vite).

## Setup

```bash
cd examples/svelte
npm install
```

Copy common assets into `public/`: `xwui-button.mjs` and `xwui/` from `examples/common/`.

## Run

```bash
npm run dev
```

Svelte treats `<xwui-button>` as a custom element; no wrapper needed.
