# XWUI + Vanilla (no framework)

Plain HTML + ES module. Proves `<xwui-button>` works with **Web Components** and no framework.

## Run

Serve this folder (e.g. `npx serve -l 3000` from this directory) then open the URL. Or use any static server; open `index.html` directly may fail due to CORS with `type="module"`.

```bash
cd examples/vanilla
npx serve -l 3000
```

Open http://localhost:3000. Assets are copied from `../common/`; see [../README.md](../README.md) to refresh from common.
